Step to create your website

  • Create a folder named My_Website
  • Inside of My_Website create a file named index.html
  • Copy the code from "HTML code for your HOME page" into index.html
  • Inside of My_Website create another file named page2.html
  • Copy the code from "HTML code for your PAGE 2" into page2.html
  • Inside of My_Website create a folder named assets
  • Get a picture of yourself & move it into your assets folder
  • Name the picture MyPic.jpg (see note1)
  • Get another picture of yourself & move it into your assets folder
  • Name this picture MyPic2.jpg (see note 1)
